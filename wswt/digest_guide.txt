How to compile Digest into Apache 2.0
Written by Dushan for RMIT WSWT Class

===== At Compile Time =====

./configure --prefix=/iweb/$USER/apache2 --enable-so --with-mpm=prefork --with-port=<yourport> --enable-ssl --enable-auth-digest
make
make install

===== With APXS =====

1: SSH into the iWeb Machine (ssh iweb1.cs.rmit.edu.au)
2: Kill your current Apache Process (pkill httpd)
3: Navigate to your home folder (cd /iweb/$USER)
4: Get the Digest source code (wget https://raw.github.com/dushankw/Projects/master/wswt/mod_auth_digest.c --no-check-certificate)
5: Navigate to the bin folder (cd /iweb/$USER/<yourapachefolder>/bin)
6: Run the following command (./apxs -i -a -c /iweb/$USER/mod_auth_digest.c)
7: Start Apache (./apachectl startssl)

===== Generate Password File =====

1: Navigate to the bin folder (cd /iweb/$USER/<yourapachefolder>/bin)
2: Run the following command (./htdigest -c /iweb/$USER/<yourapachefolder>/conf/pass_file <realm> <username>)

===== Example Config for httpd.conf =====

<Location /path/to/folder/we/want/to/protect/>
    AuthType Digest
    AuthName "<realm>"
    AuthDigestDomain /path/to/folder/we/want/to/protect/
    AuthDigestFile /iweb/<username>/<yourapachefolder>/conf/pass_file
    Require valid-user
</Location>
